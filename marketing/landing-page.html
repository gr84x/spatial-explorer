Visualize Spatial Data in 3D

Open source visualization toolkit for geospatial and 3D datasets. Built for data scientists and researchers who need to explore complex spatial relationships.

Get Started View on GitHub
import spatial_explorer as se # Load your spatial data viewer = se.Viewer(data='geospatial.parquet') # Visualize in 3D with one line viewer.plot( x='longitude', y='latitude', z='elevation', color='temperature' ) # Interactive 3D view in your browser viewer.show()

Multi-Format Support

Load GeoJSON, Shapefile, Parquet, CSV, NetCDF, or any format pandas can read. Automatic coordinate system detection.

se.load('data.geojson')
se.load('points.parquet')
se.load('grid.nc')

High Performance

GPU-accelerated rendering. Visualize millions of points in real-time with spatial indexing and level-of-detail optimization.

# 10M points, 60fps
viewer.plot(df, point_size=2)

Analysis Ready

Interactive selection, filtering, and measurement tools. Export selections back to pandas for statistical analysis.

Smart Styling

Automatic color mapping, customizable gradients, categorical and continuous colormaps. Built-in support for terrain shading.

hillshade=True)
